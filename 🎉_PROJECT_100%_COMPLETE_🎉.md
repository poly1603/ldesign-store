## 🚀 立即可用

### 构建所有包

```bash
cd packages/store
.\build-all.ps1
```

### 运行示例

```bash
# Vue 示例
cd examples/vue-example
pnpm install && pnpm dev

# React 示例
cd examples/react-example
pnpm install && pnpm dev
```

### 运行测试

```bash
cd packages/core
pnpm install && pnpm test
```
