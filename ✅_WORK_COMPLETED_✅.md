## 💡 快速示例

```typescript
// 1. 安装
pnpm add @ldesign/store-<framework>

// 2. 创建 Store
const store = createStore({
  name: 'user',
  initialState: { name: '' },
  persist: true, // 自动持久化
  cache: { maxSize: 100 } // 缓存
})

// 3. 使用
store.setName('张三')
// 刷新页面，数据自动恢复！
```
