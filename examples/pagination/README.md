# 分页列表示例

演示如何实现一个完整的分页列表功能，包括数据加载、分页控制、筛选和排序。

## 功能特性

- ✅ 分页加载数据
- ✅ 页码和页大小控制
- ✅ 数据筛选和搜索
- ✅ 排序功能
- ✅ 加载状态管理
- ✅ 错误处理
- ✅ 缓存优化
- ✅ 自动重新加载

## 安装依赖

```bash
pnpm install
```

## 运行示例

```bash
pnpm dev
```

## 主要文件

- `src/stores/pagination.ts` - 分页 Store 实现
- `src/components/PaginationList.vue` - 分页列表组件
- `src/components/PaginationControls.vue` - 分页控件组件
